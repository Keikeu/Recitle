# Recitle

**A daily song-guessing game — but the lyrics have been... transformed.**

> Can you recognise _Bohemian Rhapsody_ rewritten as a pirate shanty? Or _Blinding Lights_ retold by a caveman?

🔗 **[Play the demo →](https://karolina.place/Recitle/)**

---

## How to Play

1. **Read the verse** — you're shown a single verse of a song, rewritten in a different style (pirate talk, Shakespearean tragedy, corporate jargon, etc.)
2. **Make a guess** — type in the song title or artist you think it is
3. **Uncover more verses** — wrong guess? Another verse is revealed. Each new clue makes it a little easier
4. **Win or lose** — guess correctly before the verses run out and you win. Run out of verses without guessing right, and you lose

---

#### A new song drops every day. Have fun!
